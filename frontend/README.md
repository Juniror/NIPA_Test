# Frontend Documentation

ส้รางโดยใช้ framework react
UI นำมาจาก https://jobs.nipa.co.th/contactus

## Setup & Running

    แก้ไขไฟล์ .env.example เป็น .env (default port = 3001)
    หากมีการแก้ไขเลข port ของ backend ให้แก้ไขในไฟล์ .env ของ frontend ด้วย

    ```bash
    npm install
    npm run dev
    ```
เมื่อรันจะอยู่ที่ localhost:3001

## Usage

ส่วนการใช้งานจะมีแค่ของ Contact Us
1.  เปิดหน้าเว็บ
2.  เลื่อนลงมาที่ Contact Us
3.  กรอกข้อมูล (ข้อมูลที่มี * คือจำเป็นต้องกรอก)
4.  กดส่ง
5.  จะเด้งไปหน้า Thank You
