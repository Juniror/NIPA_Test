# Frontend Documentation

UI นำมาจาก https://jobs.nipa.co.th/contactus
ผมขอความกรุณาด้วยนะตรับ เนื่องจากว่าผมไม่ได้ไปขออนุญาติพวกภาพต่างๆเพื่อนำมาใช้งาน ผมขอโทษด้วยครับ งานนี้ไม่ได้มีจุดประสงค์เพื่อการค้าหรือการหลอกลวงใดๆทั้งสิ้น ผมทำเพื่อจุดประสงค์เพื่อการสัมภาษณ์เท่านั้น

## Usage

ส่วนการใช้งานจะมีแค่ของ Contact Us

1.  เปิดหน้าเว็บ
2.  เลื่อนลงมาที่ Contact Us
3.  กรอกข้อมูล (ข้อมูลที่มี * คือจำเป็นต้องกรอก)
4.  กดส่ง
5.  จะเด้งไปหน้า Thank You
